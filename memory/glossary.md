# Glossary

Project-specific terms, acronyms, and shorthand. Add entries as they come up so Claude (and new contributors) can decode conversations quickly.

## Format

```
**Term** — one-line definition. Optional link or pointer.
```

## Entries

**Developer Tools** — this project; a TypeScript + HTML/CSS scaffold for building small dev utilities.

**Design tokens** — the CSS custom properties in `src/style.css` (`--color-*`, `--space-*`, `--radius-*`) sourced from the Figma design.

**ADR** — Architecture Decision Record. See `memory/decisions.md`.

**Flat config** — ESLint 9's `eslint.config.js` style (replaces `.eslintrc.*`). The project uses flat config exclusively.
