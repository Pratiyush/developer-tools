# Tokens

The tables below are **auto-generated** from
`src/stories/_styles/foundations.css` via
`scripts/extract-css-tokens.ts` (SB-20 / #58). Adding a new
`--dt-*` variable to the CSS makes it appear here on the next
Storybook build — no hand-edit needed.

CI gates against drift via `pnpm tokens:check` — if the
committed snapshot disagrees with the live extraction, the build
fails.

## How to add a token

1. Edit `src/stories/_styles/foundations.css` and add the
   `--dt-*` variable under the appropriate selector
   (`:where(.dt-foundation-host)` for base, `[data-preset="…"]`
   for preset-specific).
2. Run `pnpm tokens:snapshot` to refresh
   `scripts/__snapshots__/tokens.json`.
3. Commit both the CSS change and the snapshot. CI verifies they
   stay in sync.
